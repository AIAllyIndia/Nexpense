The **Settings** section allows users to manage their account, app preferences, notifications, and support options. It can be accessed directly from the Dashboard.

![[nexpense_dashboard_overview_settings_v1.png]]

## **Accessing Settings**

1. From the Dashboard, tap the **Settings icon** (⚙️) located in the top-right corner.
    
2. This opens the **Settings** screen, where you can customize and manage various aspects of your account and the application.
    
---

## **Settings Menu Options**

The Settings screen includes the following sections:

### **1. User Profile**

- Displays your profile name and email address.
    
- Tap to view or update personal details such as profile information (if supported).
    

---

### **2. Security & Password**

- Manage your login credentials.
    
- Change your password.
    
- Helps keep your account safe and secure.
    

---

### **3. Notifications**

- Control all app-related alerts and reminders.
    
- Customize what you want to be notified about (e.g., report status updates, reminders).
    

---

### **4. App Preferences**

- Modify general app settings.
    
- Manage:
    
    - App theme (light/dark)
        
    - Currency display
        
    - Default behavior for actions
        
- Allows you to personalize the app according to your usage style.
    
![[nexpense_dashboard_overview_settings_preferences_v1.png]]


>### **a) Theme Style**
You can choose between two interface styles:
>#### **• Premium Apps**
 >- Modern rounded UI
>- Soft gradients and clean design
>- Available in **Light** and **Dark** modes
>#### **• Web Consistent**
>- Matches the look and feel of the web dashboard
>- Ideal for users who work across both platforms 
>- Available in **Light** and **Dark** modes
  >  
This allows users to pick a UI style they are most comfortable with.
>
>### **b) Brightness Mode**
>
>Choose how the app handles brightness:
>
>- **Light** → Always uses light mode
>
>- **Dark** → Always uses dark mode
>- **System** → Automatically follows the phone’s system theme
This helps users adapt the app to their preferred viewing comfort.
>
>### **c) Default Currency**
Choose the primary currency used across the app.
Examples:
>- INR (₹)
>- USD ($)
>- EUR (€)
>- GBP (£)
 The selected currency is applied to:
>- Expenses 
>- Reports
>- Approvals
>- Exported data
>### **b) Language**
Select your preferred language for the app interface.
>### **d) Brightness Mode**
Choose how the app handles brightness:
>- **Light** → Always uses light mode 
>- **Dark** → Always uses dark mode
>- **System** → Automatically follows the phone’s system theme
This helps users adapt the app to their preferred viewing comfort.
