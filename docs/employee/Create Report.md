## Create Expense Report

### Step 1: Login to Employee Portal

- Login to the **NexPense Employee Portal** using your username and password.
    
- After login, you will see the **left-side menu**.

### Step 2: Open Create Report

- From the left menu, click **Create Report**.

![[nexpense_ employee_Creport_overview_1_v1.png|300]]


### Step 3: Create New Report Page Opens

- The **Create New Report** screen will appear.
- This page is used to **group multiple expenses** under one report.

![[nexpense_ employee_Creport_overview_2_v1.png|500]]

### Enter Report Name (Required)

- In **Report Name**, enter a meaningful name.
    
- This helps managers identify your report easily.

### Select End Date (Required)

- Click the **End Date** field.
    
- Choose the **last date** of the expense period.

### Select Project (Optional)

- Click the **Project** dropdown.
    
- If your company uses projects, select the correct one.
    
- If no project is assigned, keep **No Project**.

### Select Cost Centre (Optional)

![[nexpense_ employee_Creport_overview_3_v1.png|600]]

- Click the **Cost Centre** dropdown.
    
- Select a cost centre if required by your company.
    
- Otherwise, leave it as **No Cost Centre**.
-
## Add Notes (Optional)

- In **Notes**, you can write the purpose of the report.
    
- This helps approvers understand your expenses.
## Save or Create the Report

You have two options:

### Option A: Save as Draft

- Click **Save as Draft**.
    
- The report is saved but **not submitted**.
    
- You can edit it later.
    

### Option B: Create Report

- Click **Create Report**.
    
- The report is created successfully.
    
- You can now **add expenses** to this report.

## Expense Report Details Page


## After Report Creation

- You will be redirected to the **Report Details** page.
-
![[nexpense_ employee_Creport_overview_4_v1.png]]

###  1️ Report Header (Top Section)

- Shows the **Report Name** (e.g., _Expense Report – January – Shruti_).
    
- Displays the **report date range**.
    
- Shows the **Report Status** as **Draft**.
    
- Shows **Total Amount**, currently **₹0.00** because no expenses are added yet.

### 2️ Submit Report Button

- The **Submit Report** button is disabled until at least **one expense** is added.
    
- This button is used to **send the report for approval** after adding all expenses.
### 3 Expenses List Section

- This section displays **all expenses** added to this report.
    
- Currently shows **“No expenses in this report yet”**.
    
- Confirms that the report is created but still empty.

### 4 Add Expense Button (Center)

- Click **Add Expense** to start adding expenses to this report.
    
- Opens the **Add Expense form** where you enter expense details.
