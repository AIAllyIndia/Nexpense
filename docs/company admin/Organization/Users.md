### How to Create a New User

### **Step 1: Open User Management**

1. Log in to the **NexPense Admin Dashboard**.
    
2. From the **left-side menu**, click on **Users**.
    
3. The **User Management** screen will open.
    

---

### **Step 2: Click ‚ÄúCreate User‚Äù**

![[nexpense_company_admin_select_createuser (2).png]]


1. On the **top-right corner** of the screen, click the **Create User** button.
    
2. A **Create New User** form will open.

![[nexpense_company_admin_createuserform (2).png|355]]
    

---

### **Step 3: Fill User Details**


Enter the required information in the form.

#### **Full Name**

- Enter the user‚Äôs full name
    
- Example: `John Doe`
    

#### **Email**

- Enter the user‚Äôs official email address
    
- This email will be used for login
    

#### **Password**

- Enter a temporary password for the user
    

#### **Confirm Password**

- Re-enter the same password to confirm
    

#### **Phone (Optional)**

- Enter the user‚Äôs phone number (optional)
    

---

### **Step 4: Assign Role & Access**

#### **Access Level**

- Select the user‚Äôs access level
    
- Example: **Employee**, **Manager**, **Admin**
    

This controls what the user can see and do in the system.

---

### **Step 5: Select Company Roles (Approval Workflow)**

- Choose the roles required for expense approvals
    
- Example:
    
    - Finance
        
    - HR
        
    - IT
        

üëâ These roles decide **who can approve expenses**.

---

### **Step 6: Select Department (Optional)**

- Choose the department for the user
    
- Example:
    
    - HR
        
    - Finance
        
    - IT
        

This helps in organizing users by department.

---

### **Step 7: Create the User**

1. Review all the entered details.
    
2. Click **Create User** to save.
    
3. The new user will be added to the user list.
    

---

### **Optional Actions**

- Click **Cancel** if you don‚Äôt want to create the user.
    
- You can edit user details later from the **Users list**.

### How to Import Users :

This feature allows admins to **add multiple users at once** by uploading a CSV or Excel file.
### **Step 1: Click ‚ÄúImport Users‚Äù**

![[nexpense_company_admin_importusers (2).png]]
1. On the **top-right corner**, click the **Import Users** button.
    
2. The **Import Users from CSV/Excel** window will open.

![[nexpense_company_admin_importrequiredfile (2).png|500]]
---

### **Step 2: Understand File Requirements**

Before uploading, make sure your file follows these rules:

- **Required**
    
    - Either **Name** or **Email** must be provided (at least one)
        
- **Optional**
    
    - Password (auto-generated if left empty)
        
    - Phone number
        
    - Employee ID
        
    - Role (default: Employee)
        
    - Status (default: Active)
        
    - Department Name or Department ID
        
    - Manager Name or Manager ID
        
- **Password Rule**
    
    - If provided, it must be **at least 6 characters**
        
- **Supported File Formats**
    
    - CSV
        
    - XLSX
        
    - XLS
        

üëâ Empty fields can be updated later by editing the user.

---

### **Step 3: Download Template (Recommended)**

1. Click **Download Template**.
    
2. Fill user details in the provided template.
    
3. Save the file after entering data.

![[nexpense_company_admin_users_fileformat.png]]

This helps avoid formatting errors.

---

### **Step 4: Upload the File**

1. Click **Select File**  
    **OR**  
    Drag and drop the CSV/Excel file into the upload area.
    
2. Make sure the correct file is selected.
    

---

### **Step 5: Import Users**

1. Click **Import Users**.
    
2. The system will process the file and create users automatically.

## **Approval Chain Editor**

This section allows you to create a **manual approval chain**.

## **How to Create a Manual Approval Chain**

![[nexpense_company_admin_addlevels.png]]


### **Step 1: Click ‚Äú+ Add Level‚Äù**

- Each **Level** represents one step in approval
    
- Example: Level 1 ‚Üí Manager, Level 2 ‚Üí Finance
    

---

### **Step 2: Choose Approval Type**

For each level, select one option:

#### **Sequential**

- Approvals happen **one after another**
    
- Example:
    
    1. Manager approves
        
    2. Then Finance approves
        

‚úî Use when approvals must follow an order

---

#### **Parallel**

- All selected approvers get the request **at the same time**
    
- Approval can move forward once required approvals are done
    

‚úî Use when multiple roles can approve together

---

### **Step 3: Select Approver Roles**

- Choose approver roles that match your company roles:
    
    - CE
        
    - HR
        
    - IT
        

These roles decide **who can approve expenses at this level**.

---

### **Step 4: Add More Levels (Optional)**

- Click **+ Add Level** again to create another approval step
    
- Levels are processed in order (Level 1 ‚Üí Level 2 ‚Üí Level 3)
    

---

### **Remove a Level**

- Click **Remove** to delete a specific level
    

---

## **Save or Clear Approval Chain**

### **Save Manual Override**

- Click **Save Manual Override**
    
- The custom approval chain becomes active for this user
    

---

### **Clear (fallback to matrix)**

- Removes all manual approval levels
    
- System switches back to **company approval matrix**
