Use this option to group your expenses together and submit them as one report. It helps keep everything organized and makes approval faster and easier.

**Create a New Report**  
Tap the _Create Expense Report_ button to start grouping your expenses.

![[nexpense_create_report_v1.png|205]]

This screen allows you to set up the basic details of your expense report before adding expenses.
#### **1. Report Name**

Enter a name for your report.  
Example: **“Monthly Reports – January”**

Choose a name that clearly represents the time period or purpose of the report.
#### **2. Date Range**

Select the start and end dates for the expense report:

- **Start Date**
    
- **End Date**
    

These dates define the period for which the expenses belong.
#### **3. Project (Optional)**

Choose the project associated with the report.  
Options may include:

- **No Project**
    
- **Nexpense (NXP)**
    
- **Website Redesign (WRD)**
    

Select a project only if your expenses are tied to that project.

#### **4. Cost Centre (Optional)**

Select the department or cost bucket for internal tracking.  
Options may include:

- **No Cost Centre**
    
- **Infrastructure**
    
- **Office Supplies**
    

This helps your company allocate expenses correctly during accounting.

#### **5. Description / Purpose (Optional)**

Add a short explanation about the expense report.  
Examples:

- “Monthly expenses for travel and office supplies”
    
- “Project-related meeting and transportation costs”
    
#### **6. Apply Advance Cash (Optional)**

If you have received any advance amount from the company, you can apply it here.  
You will see:

- **Available Amount**
    
- **Total Advance Balance**
    

Whatever amount you enter will be deducted from the final approved total.

#### **7. Save as Draft**

Tap **“Save Report as Draft”** to save your progress.  
You can return later to edit or add expenses before submitting.
