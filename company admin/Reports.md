## How to Use Filters in Company Reports

![[nexpense_company_admin_expenses_Reports__Filter2_v1.png|700]]


### Step 1: Search Reports (Optional)

- In **Search reports‚Ä¶**, type the report name.
    
- Matching reports will be shown.

### Step 2: Filter by Status

- Click the **All Status** dropdown.
    
- Select a status such as:
    
    - Submitted
        
    - Pending L1 / L2 / L3 / L4 / L5 Approval
        
    - Manager Approved
        
    - Approved
        
    - Rejected
        
- Reports with the selected status will be displayed.
### Step 3: Filter by Employee

- Click the **All Employees** dropdown.
    
- Select an employee name (example: _Piyush Chawhan_, _Raja Babu_).
    
- Only reports created by the selected employee will be shown.
### Step 4: Filter by Department

- Click the **All Departments** dropdown.
    
- Select a department (example: _HR_, _Marketing_, _Sales_, _Tech/IT_).
    
- Only reports from the selected department will appear.
### Step 5: Filter by Date Range

- In **Start Date**, select the beginning date.
    
- In **End Date**, select the ending date.
    
- Reports within this date range will be displayed.

![[nexpense_company_admin_expenses_Reports__Filter1_v1.png]]

### Step 6: Scroll Down to View Reports

- **Scroll down the page** to view the filtered report list.
    
- You can see:
    
    - Report Name
        
    - Employee
        
    - Department
        
    - Amount
        
    - Created Date
        
    - Status

### Report Page 

![[nexpense_company_admin_expenses_Reports__View2_v1.png|600]]


### View Report Summary (Top Section)

- Shows key information at the top:
    
    - **Total Amount** ‚Äì total expense amount in the report
        
    - **Status** ‚Äì current approval status (e.g., Pending L2 Approval)
        
    - **Expenses Count** ‚Äì number of expenses in the report
        
    - **Department** ‚Äì department linked to the report

### View Report Information

- Displays detailed report data such as:
    
    - **Employee name**
        
    - **Manager**
        
    - **Department**
        
    - **Project (if any)**
        
    - **Start Date & End Date**
        
    - **Created On & Submitted On dates**

### View Approval Timeline

- Shows the complete approval flow:
    
    - Report Created
        
    - Submitted
        
    - Manager Approval
        
    - Further approval stages (if applicable)
        
- Helps track who approved the report and when.

### View Expenses List

- Scroll down to see **all expenses** in the report.
    
- Each expense shows:
    
    - Vendor name
        
    - Category
        
    - Amount
        
    - Receipt attachment (if available)

### Export as Excel

- Click **Export Excel** (top-right).
