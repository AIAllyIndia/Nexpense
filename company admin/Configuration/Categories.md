This section lets you manage all expense categories used in your company.

##### **Where to Find It**

From the left menu, click **Configuration → Categories** to open your approval dashboard.

![[nexpense_company_admin_categories_v1.png]]
##### **What You Can Do**

- **View all categories** — each shows its name, code, description, and status.
    
- **Add Category** — create a new category using the **Add Category** button.
    
- **Edit Category** — update name, description, or status using the pencil icon.
    
- **Delete Category** — remove unwanted categories using the delete icon.
    
- **Activate/Deactivate** — mark categories as _Active_ or _Inactive_ based on company needs.
    
- **Default Categories** — system categories that cannot be deleted.

![[nexpense_company_admin_categories_overview_v1.png]]

### **Add Category**

This screen lets Company Admins create a new expense category for employees to use.

##### **What You Can Do**

- **Add Category** – Click the **Add Category** button to create a new expense category.
    
- **Enter Details** – Provide the category name, code, and description.
    
- **Set Status** – Mark the category as _Active_ or _Inactive_.
    
- **Save** – Once saved, the new category appears in the Categories list and becomes available for expense selection.

![[nexpense_company_admin_categories_add category_v1.png]]

### **Create New Category**

Use this form to create a new expense category that employees can use while adding expenses.

##### **How it works**

- **Category Name (Required):** Enter the name of the category (e.g., Travel, Meals, Office Supplies).
    
- **Category Code (Optional):** Add a short code for easier identification (e.g., TRV, MEAL, OFF).
    
- **Description (Optional):** Provide a short explanation of what the category is used for.
    
- Click **Create Category** to save it.

![[nexpense_company_admin_category_create_v1.png]]
