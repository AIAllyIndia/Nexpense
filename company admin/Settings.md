The **Company Settings** page allows admins to customize how the admin dashboard looks and how information is displayed.  
On the left menu, under **Cost Centres**, you will see **Projects**.  
Clicking on it opens the project management page.

![[nexpense_company_dashboard_settings_v1.png|250]]

Any changes you make here apply to your account and help you personalize your workspace.

![[nexpense_company_dashboard_settings_overview_v1.png]]

---

### **1. Appearance**

This section lets you choose your preferred **Theme Mode**.

#### **• Dark Mode**

A neon-style dark theme that is easier on the eyes and great for low-light environments.

#### **• Light Mode**

A clean, professional light theme suitable for well-lit workspaces.

Your selected theme is saved automatically and stays active the next time you log in.

![[nexpense_company_dashboard_settings_appearance_v1.png]]

### **2. Display Preferences**

These settings control how language and currency information appear across your dashboard.

#### **Language**

Choose the language you want the dashboard to use.  
(Example: English)

#### **3. Display Currency**

Select the currency you want amounts to be shown in.  
(Example: INR ₹)

This only affects how values are displayed—your company’s actual financial data remains unchanged.

![[nexpense_company_dashboard_settings_display_preference_v1.png]]

 **1. Light Mode Logo**
This logo is shown when users are using the dashboard in **light theme**.
**Options available:**

- **View Current Logo**  
    Shows the logo currently being used.
    
- **Delete Logo**  
    Removes the existing light mode logo.
    
- **Replace Light Logo**  
    Upload a new light-mode logo file.
    
**Supported formats:** JPG, PNG, GIF  
**Max size:** 5 MB

**2. Dark Mode Logo**

This logo appears when users switch to **dark theme**.

##### **If no dark mode logo is uploaded:**
The system will automatically use the **light mode logo** as a fallback.
**Options available:**

- **Upload Dark Logo**  
    Add a logo specifically designed for dark backgrounds.
    
**Supported formats:** JPG, PNG, GIF  
**Max size:** 5 MB

![[nexpense_company_dashboard_settings_company_logo_v1.png]]

### **4. General Company Info**

1. **Company Name**
Enter the official name of your company.  
(Shown on dashboards, reports, and notifications.)
2. **Timezone**

Choose your company’s primary timezone.  
(All logs, reports, and timestamps follow this setting.)

3. **Currency**

Select the default currency used for all spending totals and reports.

4. **Date Format**

Pick how dates should appear across the platform  
—for example: **DD/MM/YYYY** or **MM/DD/YYYY**.

5. **Financial Year Start Month**

Set the beginning of your company’s financial year.  
(Example: If you choose April, the financial year runs April → March.)

### **5. Approval Configuration**

Approval settings have been moved to a dedicated page for better clarity.  
Use the button below to create or manage approval levels and rules.

**→ Go to Approval Matrix**

**Approval Rules & Budget Controls**

Use this section to set extra approval steps based on spending limits.  
Whenever an amount crosses the threshold you define, the system will automatically send the report to the required approver.

**Examples shown:**

- **If amount exceeds ₹3,000 → Business Head approval is required**
    
- **If amount exceeds ₹5,000 → Accountant approval is required**
    

You can **add**, **edit**, or **delete** rules anytime using the buttons provided.

![[nexpense_company_dashboard_settings_general_info_config_v1.png]]

### **6. Expense Settings**

These options help you control how employees submit spending records.
1. **Require Receipt for All Expenses:** Turn this ON if every submission must include a receipt.

2. **Require Category Selection:** Force users to choose a category (e.g., Travel, Food, Office Supplies) for each record.
3. **Max File Size for Receipts:** Set the maximum allowed upload size (in MB). Example shown: **10 MB**.
4. **Allowed File Types:** Define which formats users can upload as receipts. Example: **jpg, jpeg, png, pdf**

![[nexpense_company_dashboard_settings_approvals_expenses_v1.png]]

### **7. Notifications**

Control how users receive updates and alerts from the system.
1. **Email Notifications:** Send important updates directly to users’ email inboxes.
2. **Web Push Notifications:** Show instant alerts inside the web dashboard.
3. **Daily Summary Email:** Send a daily summary of pending tasks, actions, or updates.
4. **Notify Users on Approval:** Notify users when their submission/report is approved.
5. **Notify Users on Rejection:** Alert users immediately if their submission/report gets rejected.

![[nexpense_company_dashboard_settings_notifications_v1.png]]

### **8. Save or Reset Settings**
1. **Save Settings:** Click this to apply all the changes you made.
2. **Reset to Default:** Use this if you want to revert everything back to the original settings.
