    Reports

    This Reports page helps you view, filter, and analyze expense reports easily.

    1 Open Reports Page

    • Open the app

    • Tap Reports from the bottom menu

    • You will see:

      • Search bar

      • Date filters

      • List of reports

      • Financial Year Trend chart

    ![[nexpense_reports_overview__v1.jpeg|250]]

    Search a Report

    • At the top, tap Search reports‚Ä¶

    • Type the report name (example: Expense Report or AI Scanned Receipt)

    • Matching reports will appear automatically.

    Use Date Filters

    Below the search bar, you will see date options:

    ![[nexpense_reports_overview_Filters_v1 (2).png|400]]

    • Last 3 Months ‚Üí Shows reports from last 3 months

    • This Month ‚ÜíShows reports of the current Month

    • This Year ‚Üí Shows reports of the current year

    • Custom ‚Üí Select your own date range

    Apply Custom Filter

    1. Tap Custom.

    2. Select a Financial Year or Select a From date and To date.

    3. Tap Apply Filter to update the report list.

    4. Tap Reset Filter to clear all selections.!

    ![[nexpense_reports_overview_custom_v1.png|600]]

    How To Download Report :

    ![[nexpense_reports_overview_Rdashboard_v1 (3).png||]]

    1 Select a Report

    • On the Reports page, click on the report card named ‚ÄúAI Scanned Receipt‚Äù

    2 Report Details Page Opens

    • The AI Scanned Receipt ‚Äì Details page opens automatically.

    • User can view:

      • Total Amount

      • Last Updated date

      • Approval Chain Status

      • Expense list

      • Expense Breakdown

    3 Click Download Icon

    • On the top-right corner, tap the Download icon.

    4 Choose Export Option

    • A pop-up menu opens.

    • Select one option:

      • Export as Excel

      • Export as CSV

    5 File Downloaded

    • The selected file is downloaded to the device.

    How to Delete a Report (Step by Step) :

    Step 1: Open Reports Page

    • Open the app.

    • Tap Reports from the bottom navigation bar.

    Step 2: Find the Report

    • Locate the report you want to delete

    Step 3: Click Delete Icon

    • On the selected report card, tap the Delete (üóëÔ∏è) icon.!

    ![[nexpense_reports_overview_Delete_v1.png|500]]

    Step 4: Choose an Action

    • Tap Cancel ‚Üí Report will NOT be deleted.

    • Tap Delete ‚Üí Report will be permanently deleted.

    ![[nexpense_reports_overview_Delete_2_v1.jpeg|250]]

    How Edit Page Report

    Step 1: Click Edit Icon

    • Find the report you want to update (example: AI Scanned Receipt).

    - Tap the Edit (‚úèÔ∏è) icon on the report card.

    Step 2: Edit Report Page Opens

    ![[nexpense_reports_overview_edit_v1.jpeg|500]]

    Step 3: Review Report Details

    On the Edit Report page, you can view and update:

    • Report Name

    • From Date and To Date

    • Project (Optional)

    • Cost Centre (Optional)

    • Notes / Purpose (Optional)

    ![[nexpense_reports_overview_Rate_2_v1.jpeg|300]]

    Step 4: Save Changes

    • After making changes, tap Save Changes at the bottom.

    - The app saves updates and returns you to the Reports page.

    How to View Exchange Rate Information

    Step 1: Click Info Icon (‚ÑπÔ∏è)

    - On the top-right corner of the Reports page, tap the Info (‚ÑπÔ∏è) icon.!
