    Projects

    The Projects section helps you manage all company projects linked with cost centres.
    Projects allow you to track spending, assign budgets, and involve specific stakeholders for better control. On the left menu, under Cost Centres, you will see Projects.
    Clicking on it opens the project management page.

    ![[nexpense_company_cost_center_projects_v1.png]]

    Each project card shows: 1. Project Name & Code: Helps identify the project quickly. 2. Short Description: Explains what the project is about. 3. Project Duration: Shows the start and end dates. 4. Budget: Displays the total budget assigned to the project. 5. Status (Active/Inactive): Tells you whether the project is currently active.

    Add a New Project

    Click the Add Project button on the top-right to create a new project.
    You can enter the project name, code, cost centre, stakeholders, budget, and description.

    ![[nexpense_company_cost_center_projects_add_v1 .png]]

    1. Project Name: Enter the main name of the project.
    2. Project Code (Optional): Unique short code for internal reference.
    3. Description (Optional): Explain what this project is about.
    4. Cost Centre: Select the cost centre that will sponsor or manage this project.
    5. Global Project (Optional): Enable this if the project should be visible to all company users. If not selected, you must assign specific stakeholders who can view the project.
    6. Project Stakeholders: Add people responsible for the project or who need access. You can add multiple stakeholders.
    7. Budget (Optional): Project‚Äôs total budget.
    8. Start & End Dates (Optional): Set the timeline for the project.
    9. Status: Choose whether the project is Active or Inactive.

    ![[nexpense_company_cost_center_projects_create_v1.png|505]]

    Create Project: Click Create Project to save the project and start using it.
