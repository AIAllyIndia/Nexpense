    Users

    Opening User Management :

    ![[nexpense_company_admin_userclick_2_v1.png]]

    1. On the left-side menu, you will see different options.

    2. Click on Organization → Users 👥.

    3. The User Management screen will open.

    ![[nexpense_company_admin_usermanagement (2).png]]

    Search & Filter Options

    This section helps the admin find users quickly from the user list.

    ![[nexpense_company_admin_userfilters.png]]

    Search Users

    • Use the Search users… box to find a user

    • You can search by:

      • User name

      • Email address

    👉 Start typing, and matching users will appear.

    How to Use Filters

    1. Click on any filter (Role, Department, or Status)

    2. Select the required option

    3. The user list updates automatically

    Filter Options

    You can narrow down the user list using filters.

    All Roles

    • View users based on their role

    • Examples:

      • Employee

      • Manager

      • Admin

    All Departments

    • View users from a specific department

    • Examples:

      • HR

      • Finance

      • IT

    All Status

    • Filter users by their account status

    • Options may include:

      • Active

      • Inactive

    How to Create a New User

    Step 1: Open User Management

    1. Log in to the NexPense Admin Dashboard.

    2. From the left-side menu, click on Users.

    3. The User Management screen will open.

    Step 2: Click “Create User”

    ![[nexpense_company_admin_select_createuser (2).png]]

    1. On the top-right corner of the screen, click the Create User button.

    2. A Create New User form will open.

    ![[nexpense_company_admin_createuserform (2).png|355]]

    Step 3: Fill User Details

    Enter the required information in the form.

    Full Name

    • Enter the user’s full name

    • Example: John Doe

    Email

    • Enter the user’s official email address

    • This email will be used for login

    Password

    • Enter a temporary password for the user

    Confirm Password

    • Re-enter the same password to confirm

    Phone (Optional)

    • Enter the user’s phone number (optional)

    Step 4: Assign Role & Access

    Access Level

    • Select the user’s access level

    • Example: Employee, Manager, Admin

    This controls what the user can see and do in the system.

    Step 5: Select Company Roles (Approval Workflow)

    • Choose the roles required for expense approvals

    • Example:

      • Finance

      • HR

      • IT

    👉 These roles decide who can approve expenses.

    Step 6: Select Department (Optional)

    • Choose the department for the user

    • Example:

      • HR

      • Finance

      • IT

    This helps in organizing users by department.

    Step 7: Create the User

    1. Review all the entered details.

    2. Click Create User to save.

    3. The new user will be added to the user list.

    Optional Actions

    • Click Cancel if you don’t want to create the user.

    • You can edit user details later from the Users list.

    How to Import Users :

    This feature allows admins to add multiple users at once by uploading a CSV or Excel file.

    Step 1: Click “Import Users”

    ![[nexpense_company_admin_importusers (2).png]] 1. On the top-right corner, click the Import Users button.

    1. The Import Users from CSV/Excel window will open.

    ![[nexpense_company_admin_importrequiredfile (2).png|500]]

    Step 2: Understand File Requirements

    Before uploading, make sure your file follows these rules:

    • Required

      • Either Name or Email must be provided (at least one)

    • Optional

      • Password (auto-generated if left empty)

      • Phone number

      • Employee ID

      • Role (default: Employee)

      • Status (default: Active)

      • Department Name or Department ID

      • Manager Name or Manager ID

    • Password Rule

      • If provided, it must be at least 6 characters

    • Supported File Formats

      • CSV

      • XLSX

      • XLS

    👉 Empty fields can be updated later by editing the user.

    Step 3: Download Template (Recommended)

    1. Click Download Template.

    2. Fill user details in the provided template.

    3. Save the file after entering data.

    ![[nexpense_company_admin_users_fileformat.png]]

    This helps avoid formatting errors.

    Step 4: Upload the File

    1. Click Select File
      OR
      Drag and drop the CSV/Excel file into the upload area.

    2. Make sure the correct file is selected.

    Step 5: Import Users

    1. Click Import Users.

    2. The system will process the file and create users automatically.

    Selecting Users & Bulk Actions

    This feature allows admins to select one or more users and perform actions on them at the same time.

    ![[nexpense_company_admin_bulk features.png]]

    How to Select Users

    • Click the checkbox next to a user’s name

    • You can select:

      • One user

      • Multiple users

    • The selected users will be highlighted

    Available Actions

    Activate

    • Makes the selected user(s) active

    • Active users can log in and use the system

    Deactivate

    • Temporarily disables the selected user(s)

    • Deactivated users cannot log in

    Delete

    • Permanently removes the selected user(s)

    • Use carefully (this action cannot be undone)

    Clear

    • Clears the selection

    • No users remain selected

    User Actions Menu (⋮)

    The Actions menu allows admins to perform individual actions on a specific user.

    To open it:

    • Click the three dots (⋮) in the Actions column next to a user.

    ![[nexpense_company_admin_action.png]]

    Available Options

    Edit User

    ![[nexpense_company_admin_edituser.png]]

    1. Find the user you want to update.

    2. Click the three dots (⋮) in the Actions column.

    3. Select Edit User.

    The Edit User window will open.

    ![[nexpense_company_admin_editor updateuser.png]]

    Edit User Fields Explained

    Name

    • Displays the user’s full name.

    • Update if the name needs correction.

    Email

    • Shows the registered email address.

    • Used for login and communication.

    Phone (Optional)

    • Add or update the user’s phone number.

    • This field is optional.

    Access Level

    • Select the user’s role in the system.

    • Example:

      • Employee

      • Manager

      • Admin

    This controls what the user can access.

    Company Roles (Approval Workflow)

    • Select roles used in the expense approval process.

    • Examples:

      • CE

      • HR

      • IT

    These roles decide who can approve expenses.

    Status

    • Choose the user’s current status:

      • Active → User can log in and use the system

      • Inactive → User cannot log in

    Department (Optional)

    • Assign the user to a department.

    • Helps organize users within the company.

    Save or Cancel Changes

    • Click Update User to save changes.

    • Click Cancel to close without saving.

    Reset Password

    • Sends a password reset process for the user

    • Allows the user to create a new password

    👉 Use this if a user forgets their password or cannot log in.

    Approval Chain (AI / Manual)

    • Configure how the user’s expenses are approved

    • Choose:

      • AI-based approval flow

      • Manual approval flow

    👉 Controls who approves the user’s expenses.

    Disable User

    • Temporarily disables the user account

    • The user cannot log in while disabled

    👉 Use this for employees who are on leave or no longer active.

    **Reset Password

    The Reset Password option allows an admin to help a user who has forgotten their password or cannot log in.

    Step 1: Open User Actions

    1. Go to Users in the Admin Dashboard.

    2. Find the user whose password needs to be reset.

    3. Click the three dots (⋮) in the Actions column.

    ![[nexpense_company_admin_resetpass.png]]

    Step 2: Click “Reset Password”

    1. From the menu, click Reset Password.

    2. The system will immediately process the request.

    Step 3: Success Message

    • A success notification appears:

      “Password reset email sent” ✅

    ![[nexpense_COMPANY_ADMIN_popup.png]]

    This confirms the action was successful.

    Approval Chain (AI / Manual)

    The Approval Chain feature allows admins to control how a user’s expenses are approved.

    How to Open Approval Chain

    ![[nexpense_company_admin_approval_chain.png]]

    1. Go to Users in the Admin Dashboard.

    2. Find the user.

    3. Click the three dots (⋮) in the Actions column.

    4. Select Approval Chain (AI / Manual).

    The Approval Chain window will open.

    ![[nexpense_company_admin_approvalchainwindow.png]]

    Approval Chain Editor

    This section allows you to create a manual approval chain.

    How to Create a Manual Approval Chain

    ![[nexpense_company_admin_addlevels.png]]

    Step 1: Click “+ Add Level”

    • Each Level represents one step in approval

    • Example: Level 1 → Manager, Level 2 → Finance

    Step 2: Choose Approval Type

    For each level, select one option:

    Sequential

    • Approvals happen one after another

    • Example:

      1. Manager approves

      2. Then Finance approves

    ✔ Use when approvals must follow an order

    Parallel

    • All selected approvers get the request at the same time

    • Approval can move forward once required approvals are done

    ✔ Use when multiple roles can approve together

    Step 3: Select Approver Roles

    • Choose approver roles that match your company roles:

      • CE

      • HR

      • IT

    These roles decide who can approve expenses at this level.

    Step 4: Add More Levels (Optional)

    • Click + Add Level again to create another approval step

    • Levels are processed in order (Level 1 → Level 2 → Level 3)

    Remove a Level

    • Click Remove to delete a specific level

    Save or Clear Approval Chain

    Save Manual Override

    • Click Save Manual Override

    • The custom approval chain becomes active for this user

    Clear (fallback to matrix)

    • Removes all manual approval levels

    • System switches back to company approval matrix

    Disable User

    The Disable User option allows admins to temporarily block a user’s access without deleting the account.

    How to Disable a User

    ![[nexpense_company_admin_disable_user.png]]

    Step 1: Open User Actions

    1. Go to Users in the Admin Dashboard.

    2. Find the user you want to disable.

    3. Click the three dots (⋮) in the Actions column.

    Step 2: Click “Disable User”

    1. From the menu, click Disable User.

    2. The system processes the request immediately.

    Step 3: Success Notification

    • A notification appears at the top:

      “User status updated” ✅

    This confirms the user has been disabled successfully.

    ![[nexpense_company_disable_suucesful.png]]

    Enable User

    The Enable User option allows admins to restore access for a previously disabled user.

    How to Enable a User

    Step 1: Open User Actions

    1. Go to Users in the Admin Dashboard.

    2. Find the disabled user.

    3. Click the three dots (⋮) in the Actions column.

    ![[nexpense_company_admin_enable_user.png]]

    Step 2: Click “Enable User”

    1. From the menu, click Enable User.

    2. The system processes the request immediately.

    Step 3: Success Notification

    • A confirmation message appears at the top:

      “User status updated” ✅

    This means the user has been enabled successfully.

    ![[nexpense_company_admin_enable_succesful.png]]
