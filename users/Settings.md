The **Settings** section allows users to manage their account, app preferences, notifications, and support options. It can be accessed directly from the Dashboard.

![[nexpense_dashboard_overview_settings_1_v1.png]]

## **Accessing Settings**



1. From the Dashboard, tap the **Settings icon** (⚙️) located in the top-right corner.
    
2. This opens the **Settings** screen, where you can customize and manage various aspects of your account and the application.
    
---

## **Settings Menu Options**

The Settings screen includes the following sections:

### **1. User Profile**

- Displays your profile name and email address.
    
- Tap to view or update personal details such as profile information (if supported).
    
![[nexpense_dashboard_overview_settings_preferences_profle_v1.png]]

Tap **Edit** in the top-right corner to enter edit mode.

In edit mode:

#### Editable Fields

- **Phone Number**  
    (This is typically user-editable unless restricted by admin)
    

#### Non-Editable Fields

These details are locked and can only be updated by your organization admin:

- Full Name
    
- Email Address
    
- Company
    
- Department
    
- Employee ID
    
- Role
    

Each locked field includes a small **info icon (i)** to indicate admin control.

![[nexpense_dashboard_overview_settings_preferences_profle__edit_v1.png]]

### **4. App Preferences**

- Modify general app settings.
    
- Manage:
    
    - App theme (light/dark)
        
    - Currency display
        
    - Default behavior for actions
        
- Allows you to personalize the app according to your usage style.
    
![[nexpense_dashboard_overview_settings_preferences_v1.png]]

#### **a) Theme Style**
You can choose between two interface styles:
##### **• Premium Apps**
 - Modern rounded UI
- Soft gradients and clean design
- Available in **Light** and **Dark** modes
##### **• Web Consistent**
- Matches the look and feel of the web dashboard
- Ideal for users who work across both platforms 
- Available in **Light** and **Dark** modes
This allows users to pick a UI style they are most comfortable with.

#### **b) Colors & Themes**
This section lets users **customize the color palette** used across the app.
- Modify accent colors
- Adjust gradients 
- Personalize the feel of the interface  
- Works in both theme styles
- Ideal for branding or comfort-based customization
This feature is especially useful for users who prefer a specific visual aesthetic.
	#### **1. Theme Style**
	At the top of the Colors & Themes screen, users can select the theme style they want to customize:
		**• Premium App**
			- Soft, rounded modern UI
			- Gradients and richer color transitions		    
			- Light & Dark mode options
		**• Web Consistent**
			- Matches the Nexpense web application's design
			- Cleaner, sharper UI
			- Available in Light & Dark modes
This selection determines how the chosen colors will be applied across the app.
	
	#### **2. Default Colors**
The app provides a set of pre-designed color presets under **Default Colors**, grouped into two categories.
		**a) Solid Colors**
		These are single-color options used for:
		- Buttons
		- Highlights
		- Navigation accent
		- Key UI elements
		Available solid colors include:	
		- Indigo    
		- Purple	    
		- Green    
		- Cyan
	Users can tap any color to apply it instantly.
	<br>
		**b) Gradient Colors**
	Gradient colors offer a more visually dynamic experience.
	Available options include:
		- Blue gradient  
		- Green gradient   
		- Orange-to-red gradient  
		- Indigo-to-cyan gradient
	Selecting a gradient theme applies it across supported UI components, giving a premium feel.
	#### **3. Custom Colors**
	This section allows users to craft their own unique color themes.
	
![[nexpense_dashboard_overview_settings_preferences_theme_colorsv1.png]]
#### **c) Brightness Mode**
Choose how the app handles brightness:
	- **Light** → Always uses light mode
	- **Dark** → Always uses dark mode
	- **System** → Automatically follows the phone’s system theme
	This helps users adapt the app to their preferred viewing comfort.

#### **d) Default Currency**
Choose the primary currency used across the app.
Examples:
- INR (₹)
- USD ($)
- EUR (€)
- GBP (£)
 The selected currency is applied to:
- Expenses 
- Reports
- Approvals
- Exported data
#### **e) Language**
Choose the language used for:

- Menus
    
- Buttons
    
- Labels
    
- Notifications
    

The current screenshot shows **English**, but this field will list all supported languages.

### **6. Advance Cash**
**1. Create Advance Request**
To request advance cash, fill in the following fields:
- **Amount** – Enter the advance amount you need.
- **Currency** – Choose the preferred currency.   
- **Project (Optional)** – Select a project if the advance is for specific work.
- **Cost Centre (Optional)** – Link the request to a cost centre if required.  

Once completed, tap **Create** to submit the request.

**2. My Advances**
Below the form, you can view all your previously requested advances.
This section will show:
- Pending advances 
- Approved advances 
- Rejected advances
If you haven’t requested any yet, it will display:  
**“No advances yet.”**
