This **Reports page** helps you **view, filter, and analyze expense reports** easily.

## 1 Open Reports Page

- Open the app
    
- Tap **Reports** from the bottom menu
    
- You will see:
    
    - Search bar
        
    - Date filters
        
    - List of reports
        
    - Financial Year Trend chart

![[nepense_reports_overview_1_v1.jpg|205]]

## Search a Report

- At the top, tap **Search reports…**
    
- Type the report name (example: _Expense Report_ or _AI Scanned Receipt_)
- Matching reports will appear automatically.

---

## Use Date Filters

Below the search bar, you will see date options:

![[nepense_reports_overview_Filters_1_v1.png|350]]


- **Last 3 Months** → Shows reports from last 3 months

- **This Month** →Shows reports of the current Month
    
- **This Year** → Shows reports of the current year
    
- **Custom**  → Select your own date range


### Apply Custom Filter

1. Tap **Custom**.
    
2. Select a **Financial Year**   **or**  Select a **From** date and **To** date.

3. Tap **Apply Filter** to update the report list.

4.  Tap **Reset Filter** to clear all selections.!


    ![[nepense_dashboard_overview_custom_v1.png|700]]

---

### 1 Select a Report 

- On the Reports page, **click on the report card** named **“AI Scanned Receipt”**
### 2 Report Details Page Opens

- The **AI Scanned Receipt – Details page** opens automatically.
    
- User can view:
    
    - Total Amount
        
    - Last Updated date
        
    - Approval Chain Status
        
    - Expense list
        
    - Expense Breakdown
    
### 3 Click Download Icon

- On the **top-right corner**, tap the **Download icon**.


### 4 Choose Export Option

- A pop-up menu opens.
    
- Select one option:
    
    - **Export as Excel**
        
    - **Export as CSV**


### 5 File Downloaded

- The selected file is downloaded to the device.
